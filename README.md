# YuanBapAPI
一个数字货币价格行情跟踪项目,采用MVP + RxJava + Retrofit,加入单元测试,采用的是元宝网公开API

## IDEA来源以及用到的API
这个项目是来自于我想观察比特币当日的行情和历史行情来做的一个项目,借用的是[元宝网](https://www.yuanbao.com/coins)的API,当然中间准备加几个比较有用的API
(当前确定的有电影和天气).
